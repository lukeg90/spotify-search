# spotify-search

## Key Features

  * Makes Ajax request to Spotify API for 20 albums or artists at a time
  * If the query `?scroll=infinite` is added to the URL, new results will be loaded automatically when scrolling
  
# Technologies

  * JavaScript
  * Ajax
  * HTML/CSS
  * Handlebars
